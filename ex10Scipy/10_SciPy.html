Ziele¶

Ziel dieser Einheit ist es

  • zu lernen, welche Möglichkeiten SciPy in der wissenschaftlichen Programmierung bietet
  • beispielhaft häufig verwendete Algorithmen kennenzulernen

SciPy ist eine Sammlung von mathematischen Algorithmen und Komfortfunktionen, die auf NumPy aufbauen.

SciPy ist in Unterpakete organisiert, die verschiedene wissenschaftliche Bereiche abdecken. Einige Beispiele:

  • constants: Physikalische und mathematische Konstanten
  • fftpack: Fast-Fourier-Transformationen
  • integrate: Integration und Löser für gewöhnliche Differentialgleichungen
  • interpolate: Interpolation und Splines
  • linalg: Lineare Algebra
  • signal: Signalverarbeitung
  • spatial: Räumliche Datenstrukturen und Algorithmen
  • stats: Statistische Verteilungen und Funktionen

Da SciPy auf NumPy aufbaut, können für die grundlegende Verwaltung von Arrays NumPy-Befehle verwendet werden:

In [2]:
import numpy as np
np.zeros(3)
Out[2]:
array([0., 0., 0.])

Lineare Algebra (scipy.linalg)¶

SciPy bietet sehr schnelle/effiziente Rechenmethoden für lineare Algebra. scipy.linalg enthält alle Funktionen von numpy.linalg und einige weitere für fortgeschrittene Anwendungen. Je nach Art der NumPy-Installation kann es sein, dass die SciPy-Varianten schneller sind.

Grundlegende Funktionen¶

Inverse einer Matrix mit inv:

In [67]:
import numpy as np
from scipy import linalg

A = np.array([[1, 3, 5], [2, 5, 1], [2, 3, 8]])
linalg.inv(A)
Out[67]:
array([[-1.48,  0.36,  0.88],
       [ 0.56,  0.08, -0.36],
       [ 0.16, -0.12,  0.04]])

Lineare Gleichungssysteme Lösen mit solve:

$$ \begin{split} x& + 3y& \;+\; 5&z = 10 \\ 2x& + 5y& \;+\; &z = 8 \\ 2x& + 3y& \;+\; 8&z = 3 \end{split} $$

In [77]:
import numpy as np
from scipy import linalg

linalg.solve(A, b)

Integration (scipy.integrate)¶

Dieses Unterpaket stellt verschiedene Integrationstechniken zur Verfügung, unter anderem einen Integrator für gewöhnliche Differentialgleichungen.

Allgemeine Integration¶

Allgemeine Integration erfolgt mit der Funktion quad:

quad(integrand, lowerLimit, upperLimit, args=(arguments))

Beispiel: Lösen des folgenden Integrals

$$ I(a,b) = \int_0^1 a x^2 + b \; dx $$

In [34]:
from scipy.integrate import quad

def integrand(x, a, b):
    return a * x**2 + b

a, b = 2, 1
I, abserr = quad(integrand, 0, 1, args=(a, b))
print(I)

1.6666666666666667

Anfangswertprobleme¶

Lösen eines Anfangswertproblems (Initial value problem) mit der Funktion solve_ivp:

solve_ivp(function, integrationInterval, initialState)

Beispiel: Exponentieller Verfall

$$ \frac{dy}{dt} = -0.5 \; y $$

In [8]:
from scipy.integrate import solve_ivp
import matplotlib.pyplot as plt

def exponential_decay(t, y):
    return -0.5 * y

sol = solve_ivp(exponential_decay, [0, 10], [2, 4])
fig = plt.figure()
plt.plot(sol.t, sol.y[0])
plt.plot(sol.t, sol.y[1]);

Interpolation (scipy.interpolate)¶

Eindimensionale Interpolation¶

Eindimensionale Interpolation erfolgt mit der Funktion interp1d:

f = interp1d(x, y, kind)

$x$ und $y$ sind Werte, die eine Funktion $f$ via $y = f(x)$ approximieren.

In [7]:
from scipy.interpolate import interp1d
import matplotlib.pyplot as plt

x = np.linspace(0, 10, num=11, endpoint=True)
y = np.cos(-x**2/9.0)
f_linear  = interp1d(x, y)
f_cubic   = interp1d(x, y, kind='cubic')
f_nearest = interp1d(x, y, kind='nearest')

fig = plt.figure()
xnew = np.linspace(0, 10, num=1001, endpoint=True)
plt.plot(x, y, 'o', xnew, f_linear(xnew), '-', xnew, f_cubic(xnew), '--', xnew, f_nearest(xnew), ':')
plt.legend(['data', 'linear', 'cubic', 'nearest'], loc='best');

Mehrdimensionale Interpolation¶

Mit der Funktion griddata können unstrukturierte, mehrdimensionale Daten interpoliert werden.

interpolationValues = griddata(dataPoints, dataValues,
                               interpolationPoints, method)

Beispiel: 1000 bekannte Punkte der zweidimensionalen Funktion

$$ f(x,y) = x \;(1-x) \;\cos(4 \pi x) \;\sin^2 (4 \pi y^2) $$

In [6]:
import numpy as np
from scipy.interpolate import griddata
import matplotlib.pyplot as plt

def func(x, y):
    return x*(1-x)*np.cos(4*np.pi*x) * np.sin(4*np.pi*y**2)**2

grid_x, grid_y = np.mgrid[0:1:100j, 0:1:200j]
points = np.random.rand(1000, 2)
values = func(points[:,0], points[:,1])

grid_z = griddata(points, values, (grid_x, grid_y), method='cubic')

fig = plt.figure()
plt.subplot(121)
plt.imshow(func(grid_x, grid_y).T, extent=(0,1,0,1), origin='lower')
plt.plot(points[:,0], points[:,1], 'r.', ms=1)
plt.title('Original')
plt.subplot(122)
plt.imshow(grid_z.T, extent=(0,1,0,1), origin='lower')
plt.title('Interpolated');

Räumliche Algorithmen (scipy.spatial)¶

Delaunay-Triangulation (Dreiecke aus Punkten)

In [9]:
from scipy.spatial import Delaunay
import matplotlib.pyplot as plt

points = np.array([[-0.3, -0.1], [0, 1.1], [1, 0], [1, 0.8], [0.6, 0.4]])
triangles = Delaunay(points)

fig = plt.figure()
plt.triplot(points[:,0], points[:,1], triangles.simplices)
plt.plot(points[:,0], points[:,1], 'o');

Konvexe Hülle (Umschließende Hülle um Punkte)

In [10]:
from scipy.spatial import ConvexHull
import matplotlib.pyplot as plt

points = np.random.rand(30, 2)   # 30 random points in 2-D
hull = ConvexHull(points)

fig = plt.figure()
plt.plot(points[:,0], points[:,1], 'o')
for simplex in hull.simplices:
    plt.plot(points[simplex,0], points[simplex,1], 'k-');

Signalverarbeitung (scipy.signal)¶

Linearen Trend aus einer Datenreihe entfernen mit detrend:

In [11]:
import numpy as np
from scipy import signal
import matplotlib.pyplot as plt

t = np.linspace(0, 5, 100)
x = t + np.random.normal(size=100)

x_detrended = signal.detrend(x)

fig = plt.figure()
plt.plot(t, x, label="x")
plt.plot(t, x-x_detrended, label="trend")
plt.plot(t, x_detrended, label="x_detrended")
plt.legend(loc='best');
